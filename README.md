# Task 1: Understanding the Dataset
Dataset Overview

This dataset contains monthly business performance information collected from multiple retail stores. The objective of the analysis is to identify the factors that influence monthly profit using regression analysis. The dataset includes both numerical and categorical variables representing sales performance, customer behavior, operational efficiency, marketing activities, and store characteristics.

## 1. Dependent Variable

The target variable for the regression analysis is:

monthly_profit

This variable represents the monthly profit generated by each retail store and serves as the outcome that the regression models aim to predict.

## 2. Potential Independent Variables

The following variables are considered potential predictors of monthly profit:

Numerical Variables
monthly_sales
customer_rating
holiday_flag
competitor_distance_km
inventory_availability_pct
staff_count
avg_discount_pct
footfall
marketing_spend
Categorical Variables
store_type
region
month

These variables represent operational, customer, marketing, seasonal, and geographical factors that may influence store profitability.

## 3. Numerical Variables
Variable	Description
monthly_sales	Total monthly sales generated by the store
customer_rating	Average customer satisfaction rating
holiday_flag	Indicates whether the month includes a holiday period (0/1)
competitor_distance_km	Distance to the nearest competitor (km)
inventory_availability_pct	Percentage of inventory available for sale
staff_count	Number of employees working in the store
avg_discount_pct	Average discount percentage offered
footfall	Total number of customer visits
marketing_spend	Monthly marketing expenditure
monthly_profit	Monthly profit (Dependent Variable)
## 4. Categorical Variables
Variable	Description
store_type	Type/category of retail store
region	Geographic region where the store operates
month	Month of the recorded business activity

These variables will require dummy variable encoding before being included in the multiple regression model.

## 6. Variables Not Useful for Regression

The following variable is not considered a useful predictor:

Variable	Reason
store_id	Unique identifier used only for record identification. It does not carry predictive business information and will be excluded from regression analysis.
Data Preparation Summary

Before building the regression models, the following preprocessing steps will be performed:

Handle missing values in competitor_distance_km and customer_rating.
Convert categorical variables into dummy variables.
Exclude store_id from model development.
Preserve the original dataset and perform all transformations on a cleaned version of the data.

This prepared dataset will be used for both simple and multiple linear regression analyses to identify the business factors most strongly associated with monthly profit.
