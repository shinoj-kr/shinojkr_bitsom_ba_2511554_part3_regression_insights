# Task 1: Understanding the Dataset

## Dataset Overview
This dataset captures **monthly business performance** across multiple retail stores. The objective is to identify the factors that influence **monthly profit** using regression analysis.  
It includes both **numerical** and **categorical** variables representing sales performance, customer behavior, operational efficiency, marketing activities, and store characteristics.

---

## 1. Dependent Variable
**Target Variable:** `monthly_profit`  
This represents the monthly profit generated by each store and serves as the outcome variable for all regression models.

---

## 2. Potential Independent Variables
The following variables are considered potential predictors of monthly profit:

### **Numerical Variables**
- `monthly_sales`  
- `customer_rating`  
- `holiday_flag`  
- `competitor_distance_km`  
- `inventory_availability_pct`  
- `staff_count`  
- `avg_discount_pct`  
- `footfall`  
- `marketing_spend`

### **Categorical Variables**
- `store_type`  
- `region`  
- `month`

These variables capture operational, marketing, seasonal, and geographical influences on store profitability.

---

### 3. Numerical Variables Description

| Variable | Description |
|-----------|-------------|
| `monthly_sales` | Total monthly sales generated by the store |
| `customer_rating` | Average customer satisfaction rating |
| `holiday_flag` | Indicates whether the month includes a holiday period (0 = No, 1 = Yes) |
| `competitor_distance_km` | Distance to the nearest competitor (in km) |
| `inventory_availability_pct` | Percentage of inventory available for sale |
| `staff_count` | Number of employees working in the store |
| `avg_discount_pct` | Average discount percentage offered |
| `footfall` | Total number of customer visits |
| `marketing_spend` | Monthly marketing expenditure |
| `monthly_profit` | Monthly profit (dependent variable) |

---

## 4. Categorical Variables Description

| Variable | Description |
|-----------|-------------|
| `store_type` | Type/category of retail store |
| `region` | Geographic region where the store operates |
| `month` | Month of the recorded business activity |

These variables will be converted into **dummy variables** before inclusion in the multiple regression model.
