# Task 1: Understanding the Dataset

## Dataset Overview

This dataset contains monthly operational and business performance data for multiple retail stores. It includes information related to sales, marketing activities, customer behavior, inventory management, staffing, store characteristics, and profitability.

The primary objective of this project is to identify the key factors associated with **monthly sales** using regression analysis and provide business recommendations based on the findings.

---

## 1. Dependent Variable

**Target Variable**

- **monthly_sales**

Monthly sales represent the total sales generated by each store during a month and will be used as the dependent (response) variable in both simple and multiple regression models.

---

## 2. Potential Independent Variables

The following variables are potential predictors of monthly sales:

### Numerical Variables

- marketing_spend
- footfall
- avg_discount_pct
- staff_count
- inventory_availability_pct
- competitor_distance_km
- holiday_flag
- customer_rating
- monthly_profit

### Categorical Variables

- region
- store_type
- month

These variables represent marketing, operational, customer, seasonal, and geographical factors that may influence monthly sales.

---

## 3. Numerical Variables

| Variable | Description |
|-----------|-------------|
| marketing_spend | Monthly marketing expenditure |
| footfall | Total customer visits during the month |
| avg_discount_pct | Average discount percentage offered |
| staff_count | Number of employees in the store |
| inventory_availability_pct | Percentage of inventory available for sale |
| competitor_distance_km | Distance to the nearest competitor (km) |
| holiday_flag | Indicates whether the month includes a holiday period (0 = No, 1 = Yes) |
| customer_rating | Average customer satisfaction rating |
| monthly_sales | Total monthly sales (Dependent Variable) |
| monthly_profit | Monthly profit generated by the store |

---

## 4. Categorical Variables

| Variable | Description |
|-----------|-------------|
| region | Geographic region where the store operates |
| store_type | Type or category of retail store |
| month | Month of business activity |

These variables will be converted into dummy variables before they are included in the multiple regression model.

---

## 5. Variables Requiring Cleaning or Transformation

| Variable | Issue | Recommended Action |
|-----------|-------|--------------------|
| competitor_distance_km | Missing values | Impute using mean or median |
| customer_rating | Missing values | Impute using mean or median |
| region | Categorical variable | Convert into dummy variables |
| store_type | Categorical variable | Convert into dummy variables |
| month | Categorical variable | Convert into dummy variables if included in the model |

No duplicate records or inconsistent column names were identified during the initial review of the dataset.

---

## 6. Variables Not Useful for Regression

| Variable | Reason |
|-----------|--------|
| store_id | Unique identifier with no predictive value for regression analysis |

---

## Summary

The dataset contains a combination of numerical and categorical variables suitable for regression analysis. After handling missing values and converting categorical variables into dummy variables, the dataset will be used to build simple and multiple linear regression models to identify the factors most strongly associated with monthly sales.
